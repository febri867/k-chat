## Codes Structure

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).<br>
And used following libs & dependencies:
- `redux` & `react-redux`
- `redux-thunk`
- `redux-logger`
- `bootstrap UI framework`
